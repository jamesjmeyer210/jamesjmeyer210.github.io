Development Projects

Moria
In Progress
Moria is a reverse proxy JWT authentication service written in Rust that provides group based permissions on endpoints. It is extremely high performant with a very low memory footprint and exceptionally fast throughput. The objective is for the proxy to be able to protect the most highly requested websites.
